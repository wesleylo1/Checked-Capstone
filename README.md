# "Checked!" Capstone Project

## About

"Checked!" is a full stack web application that allows users to create and save checklists for different occasions in their account.

## MVP

- Create a user account
- Log into the account
- Log out of the account
- Create a new checklist or choose from a few premade ones
- Edit the checklist by editing or deleting tasks
- Delete the checklist
